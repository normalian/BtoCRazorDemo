## 本プロジェクトは ASP.NET MVC4 を利用したショッピングサイト形式のサンプルプロジェクトです。

## 前提条件
 * Visual Studio 2012(Testing Framework を動作させる場合、Standard Edition 以上が必要)
 * ASP.NET MVC4
 * SQL Server 2012 Express LocalDB
   * 以下を参考に、 SQL Server 2012 Express LocalDB を利用する
   * http://www.microsoft.com/ja-jp/download/details.aspx?id=29062
 * Moq

## 備考
 * ログインフォームが存在しますが、パスワードは特にチェックしていません。ユーザ名のみを見て認証トークンを発行しています
 * 商品データはすべて B2CDb.mdf に格納されています
